## Background color

Use a `backgroundColor` to change the background color of the avatar.

<Example Component={AvatarItemBackgroundColor} packageName="@atlaskit/avatar" />

## States

### Disabled

Use `isDisabled` to put the avatar into the disabled state. This will make the avatar non-interactive.

<Example Component={AvatarItemIsDisabled} packageName="@atlaskit/avatar" />

## Text

### Primary text

Use `primaryText` to style the avatar text as primary text.

<Example Component={AvatarItemPrimaryText} packageName="@atlaskit/avatar" />

### Secondary text

Use `secondaryText` to style the avatar text as secondary text.

<Example Component={AvatarItemSecondaryText} packageName="@atlaskit/avatar" />

### Composing text

`primaryText` and `secondaryText` can be composed together.

<Example Component={AvatarItemText} packageName="@atlaskit/avatar" />

## Truncation

By default, overflowing text is truncated if it exceeds the width of its container. Use `isTruncationDisabled` to disable this.

<Example
  Component={AvatarItemIsTruncationDisabled}
  packageName="@atlaskit/avatar"
/>
