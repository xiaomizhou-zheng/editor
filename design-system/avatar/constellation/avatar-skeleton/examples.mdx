## Appearance

### Circle

The default appearance. Use `appearance="circle"` for a circular skeleton.

<Example Component={AvatarSkeletonCircle} packageName="@atlaskit/avatar" />

### Square

Use `appearance="square"` for a square skeleton.

<Example Component={AvatarSkeletonSquare} packageName="@atlaskit/avatar" />

## Size

Use `size` to define the size of the skeleton.

<Example Component={AvatarSkeletonSize} packageName="@atlaskit/avatar" />

## Color

### Default

By default, a skeleton inherits the color of its parent container.

<Example
  Component={AvatarSkeletonColorDefault}

/>

### Specific color

Set the color of a skeleton by defining the `color` property.

<Example Component={AvatarSkeletonColor} packageName="@atlaskit/avatar" />
