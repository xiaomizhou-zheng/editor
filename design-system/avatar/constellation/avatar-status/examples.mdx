## Status

Use the avatar status component to show information about an avatar using a small icon in the top right of the avatar.

### Approved

Indicates an approved status.

<Example Component={AvatarStatusApproved} packageName="@atlaskit/avatar" />

### Declined

Indicates a declined status.

<Example Component={AvatarStatusDeclined} packageName="@atlaskit/avatar" />

### Locked

Indicates a locked status. This may show if an entity is private (like a locked space in Confluence).

<Example Component={AvatarStatusLocked} packageName="@atlaskit/avatar" />

## Using avatar

The `Status` can be passed directly to `Avatar` instead of using the `status` prop.

<Example Component={AvatarStatusUsingAvatar} packageName="@atlaskit/avatar" />

## Border color

By default, a status has a white border color. Use borderColor to override the default border color of the status indicator. This accepts any color that the CSS property `border-color` accepts.

<Example Component={AvatarStatusBorderColor} packageName="@atlaskit/avatar" />

## Size

By default, statuses will stretch to fill their parents. Try resizing the wrapping `div` below to see this in action.

It's best to always have a wrapping `div` around `Status` components when consuming them separately to `Avatar`.

<Example Component={AvatarStatusSize} packageName="@atlaskit/avatar" />
