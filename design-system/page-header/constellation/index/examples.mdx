## Default

Use a default page header for a title underneath [breadcrumbs](/components/breadcrumbs/examples).

The header automatically wraps if the text is too long. This can be disabled with `truncateTitle`. However, this is not recommended (see [accessibility](/components/page-header/usage#accessibility)).


<Example Component={PageHeaderDefault} packageName="@atlaskit/page-header" />

## Complex

Page headers can include [breadcrumbs](/components/breadcrumbs/examples), which appear above the header. You can also pass [buttons](/components/button/examples) into `actions` â€” these appear to the right of the header.

To help users refine the page content on a more granular level, a  [filter bar](/components/textfield/examples) and [search dropdown](/components/dropdown-menu/examples) component can be added to the `bottomBar` below the header title.
