## Appearance

A group of avatars can be displayed together in a stack or a grid.

### Stack

Stacked avatar groups can contain up to five avatars and should only be used with small or medium sized avatars.

<Example Component={AvatarGroupStack} packageName="@atlaskit/avatar-group" />

### Grid

Avatar groups displayed in a grid can contain up to 11 avatars spread across its rows.

<Example Component={AvatarGroupGrid} packageName="@atlaskit/avatar-group" />

## Max count

The maximum number of avatars allowed in the list can be customized.

<Example Component={AvatarGroupMaxCount} packageName="@atlaskit/avatar-group" />

## Border color

The color of the border around the avatar. Any color that the CSS `border-color` property accepts can be used.

<Example
  Component={AvatarGroupBorderColor}
  packageName="@atlaskit/avatar-group"
/>

## Overrides

Custom components can be passed in via the `overrides` prop.

For example, you can add extra behavior into the overflow menu using the `overrides`
prop. Click the overflow avatar and you'll see a load more button
powered by our custom `AvatarGroupItem`.

<Example
  highlight="33-50"
  Component={AvatarGroupOverrides}
  packageName="@atlaskit/avatar-group"
/>
