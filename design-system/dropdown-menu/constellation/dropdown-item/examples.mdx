## Multiline

Use `shouldTitleWrap` to allow the primary content to wrap over multiple lines.
Defaults to `true`.

<Example
  Component={DropdownItemMultiline}
  packageName="@atlaskit/dropdown-menu"
/>

Use `shouldDescriptionWrap` to allow the description to wrap over multiple lines.
Defaults to `true`.

## States

### Disabled

Use `isDisabled` to make the element appear disabled.
This will remove interactivity and hide it from screen readers.

<Example
  Component={DropdownItemDisabled}
  packageName="@atlaskit/dropdown-menu"
/>

## With elements

### Elem before

Use `elemBefore` to show an element before the dropdown item text.
Generally this should be an [icon](/components/icon).
