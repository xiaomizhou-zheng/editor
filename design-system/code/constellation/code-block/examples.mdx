## Default

A code block highlights an entire block of code and keeps the formatting.

<Example Component={CodeBlock} packageName="@atlaskit/code" />

## Line numbers

A code block can be displayed with or without line numbers. Line numbers are shown by default.

<Example Component={CodeBlockLineNumbers} packageName="@atlaskit/code" />

## Line highlighting

Lines in a code block can be highlighted.

- To highlight one line: `highlight="3"`
- To highlight a group of lines: `highlight="1-5"`
- To highlight multiple groups: `highlight="1-5,7,10,15-20"`

<Example Component={CodeBlockLineHighlight} packageName="@atlaskit/code" />

## Theme

Code block is displayed in a `light` theme by default. Use `dark` to display code in a dark theme.

<Example Component={CodeBlockHighlightColors} packageName="@atlaskit/code" />
