## Illustrations

### Image URL

You can display an image by supplying the `imageUrl` prop. This url will be passed directly into the `src` attribute of an `img` component.

<Example Component={EmptyStateImageUrl} packageName="@atlaskit/empty-state" />

### Render image

An alternate approach to displaying an image is to use the `renderImage` prop. This render prop approach will only be used if there is no `imageUrl` prop supplied.

<Example Component={EmptyStateRenderImage} packageName="@atlaskit/empty-state" />

### Image dimensions

Setting the `imageWidth` and `imageHeight` props can be useful to prevent your layout from reflowing when images load in.

If you have used the `imageUrl` approach, the `imageWidth` and `imageHeight` props set the `width` and `height` attributes on the underlying `img` element. If you have used the `renderImage` prop, they will be passed in as props to the render function.

If you are resizing a spot illustration to try and squeeze it into a layout, it's a good time to ask whether you should include the image at all.

<Example Component={EmptyStateImageDimensions} packageName="@atlaskit/empty-state" />

### Maximum image dimensions

Applying `maxImageHeight` and `maxImageWidth` works in the same way.

<Example Component={EmptyStateImageMaxDimensions} packageName="@atlaskit/empty-state" />
